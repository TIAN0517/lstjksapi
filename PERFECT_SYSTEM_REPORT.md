## 🚀 部署和使用

### 快速部署
```bash
# 1. 克隆项目
git clone https://github.com/your-username/BossJy-Cn.git
cd BossJy-Cn

# 2. 运行部署脚本
chmod +x deploy-production.sh
./deploy-production.sh

# 3. 访问系统
# 前端: http://localhost:9001
# API: http://localhost:18001
# 监控: http://localhost:3001
```

### 开发环境
```bash
# 启动开发环境
docker-compose up -d

# 运行测试
cd services/fastapi
python run_tests.py --coverage

# 查看日志
docker-compose logs -f
```
